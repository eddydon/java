Units View


Available templates:

  • DHXPlanner.templates.day_date(String format) - the date in the header of the view.
  • DHXPlanner.templates.event_class(String className) - the css style for the event container.
  • DHXPlanner.templates.event_header(String format) - the header of an event.
  • DHXPlanner.templates.event_text(String text) - the text of an event.
  • DHXPlanner.templates.hour_scale(String format) - the items of the Y-Axis.
  • DHXPlanner.templates.week_date_class(String className) - the css style for a day cell.
