How to use the license key?

Java Planner is distributed under Commercial and Enterprise Licenses. Once you buy a license you get the license key (over email) that you need to use to activate the library.

 
Please, don't mix up the license key with the license number. These are different things.

To activate the scheduler you need the license key. In the confirmation letter the key looks like this:

How can you activate the key?

The key is activated directly in the project(application) by calling method DHXPlanner.setLicenseKey(key). You should call the method in the main .jsp file where you initialize the scheduler: 

//Sampleplanner.jsp
String getJava Planner (HttpServletRequest request) throws Exception {
        DHXPlanner.setLicenseKey("yourLicenseKey");// activates the license key      
	DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
        ...
	return planner.render();
}

Important tips!

  • the DHXPlanner.setLicenseKey() method must be called before the DHXPlanner.render() method.
  • The method is called once in an application.
