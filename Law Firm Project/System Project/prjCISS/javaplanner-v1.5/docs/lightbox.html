Lightbox

In this chapter you'll know how to:

  • Add a control to the lightbox;
  • Create fully custom lightbox.

How to add a control to the lightbox

Generally, to add a control to the lightbox you should use method DHXPlanner.lightbox.add(DHXLightboxItem control, Integer index): 

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
DHXLightboxSelect select = new DHXLightboxSelect("event_type", "Event type");

The library provides the following controls classes:

  • DHXLightboxMiniCalendar - a date input with popup calendar;
  • DHXLightboxCheckbox - a check box;
  • DHXLightboxMultiselect - a multi-select list box;
    • DHXLightboxSelectOption - a select option of the list box.
  • DHXLightboxRadio - a set of radio buttons;
    • DHXLightboxSelectOption - a single radio button.
  • DHXLightboxSelect - a dropdown list;
    • DHXLightboxSelectOption - a select option of the dropdown list.
  • DHXLightboxText - a text input;
  • DHXLightboxTime - a control area containing 2 date inputs for setting start and end times;
  • DHXLightboxReccuringBlock - a control area used to set values for reccuring events.
DHXLightboxCheckbox check = new DHXLightboxCheckbox("highlighting", "Important");
check.setMapTo("textColor");
check.setCheckedValue("red");
 
planner.lightbox.add(check);
planner.lightbox.add(new DHXLightboxText("text", "Description"));

By default, the planner contains 2 controls:

  • time - (DHXLightboxTime) the start (when an event is scheduled to begin) and end (when an event is planner to be completed) times.
  • text - (DHXLightboxText) the text of an event.

Beware, as soon as you add some custom control, the default controls will be removed. To add some new control and hold the default ones, add the following code to the action: 

planner.lightbox.addDefaults();

Dropdown controls

Here is a common technique for adding select(multiselect, radio) controls to the lightbox: 

DHXLightboxSelect select = new DHXLightboxSelect("textColor", "Priority");
select.setServerList("textColor");
planner.lightbox.add(select);

or 

DHXLightboxSelect select = new DHXLightboxSelect("textColor", "Priority");
select.addOption("gray", "Low");
select.addOption("blue", "Medium");
select.addOption("red", "High");
 
planner.lightbox.add(select);

Note, options take 2 properties: key (the id of an option) and label ( the text label of an option).

Fully Custom Lightbox

To define a custom lightbox you should apply 2 steps:

  1. Create a view containing a form with the desired controls;
  2. With method DHXPlanner.lightbox.SetExternalLightboxForm(String viewName), 'load' the view into an iframe inside the lightbox popup.

    DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...			
DHXExternalLightboxForm box = 
         planner.lightbox.setExternalLightboxForm("./custom_editor.jsp", 296, 140);   
 
box.setClassName("custom_lightbox");

View sample

Note, you should define 2 methods for you custom form:

  • setValues(object) - takes an event object and sets values for the controls. the planner calls the method (after the view is loaded) and passes an edited event to it.
  • object getValues() - takes values from the inputs (controls) and returns an event object. the planner calls the method to get an event object, update view data and send updated data to the server.
//custom_editor.jsp
<style>
#form1 {
	font-family: Tahoma;
	font-size: 10pt;
}
#form1 label {
	width: 100px;
	display: inline-block;
}
</style>
<div>
	<form id='form1'>
		<label form="event_text">Text</label>
                <input id="event_text" name="text" type="text"/><br />
		<label for="event_start_date">From</label>
                <input id="event_start_date" name="start_date"  type="text"/><br />
		<label for="event_end_date">To</label>
                <input id="event_end_date" name="end_date" type="text"/><br />
		<input id="Hidden1" name="id" type="hidden"/>
	</form>
	<div>
		<input type="button" value="Save" onclick="lightbox.save()" />
		<input type="button" value="Cancel" onclick="lightbox.close()" />
		<input style="margin-left:86px" type="button" value="Delete" onclick="lightbox.remove()"/>
	</div>
</div>
<script>
	getValues = function () {
		var ev = {};
		var inputs = document.body.getElementsByTagName('input');
 
		for (var i = 0; i < inputs.length; i++) {
			if (inputs[i].type == "button") {
				continue;
			}
			var name = inputs[i].getAttribute('name');
			if (name == "start_date" || name == "end_date")
				ev[name] = parent.planner.templates.api_date(inputs[i].value);
			else
				ev[name] = inputs[i].value;
		}
		return ev;
	};
 
	setValues = function (obj) {
		var inputs = document.body.getElementsByTagName('input');
		for (var i = 0; i < inputs.length; i++)
			if (inputs[i].type != "button") {
				var name = inputs[i].getAttribute('name');
				if (name == "start_date" || name == "end_date")
					inputs[i].value = parent.planner.date.date_to_str(parent.planner.
                                        config.api_date)(obj[name]);
				else
					inputs[i].value = obj[name];
			}
	};
 
</script>
  • parent.planner.templates.api_date - the function converts a specified string to a date object.
  • parent.planner.date.date_to_str(parent.planner.config.api_date) - converts a date object to a string of the format supported by the planner.

The method SetExternalLightboxForm()

Method DHXPlanner.lightbox.setExternalLightboxForm(String viewName) can be called in 3 ways with a different number of parameters: 

DHXExternalLightboxForm box = planner.lightbox.setExternalLightboxForm("./custom_editor.jsp", 296, 140);
  • the path to a view that you want to load;
  • the width of the lightbox;
  • the height of the lightbox.

DHXExternalLightboxForm box = planner.lightbox.setExternalLightboxForm("./custom_editor.jsp","classname");
  • the path to a view that you want to load;
  • the css class that will be applied to the lightbox.

DHXExternalLightboxForm box = planner.lightbox.setExternalLightboxForm("./custom_editor.jsp");
  • the path to a view that you want to load.


The method returns an DHXExternalLightboxForm object that has the following methods:

  • DHXExternalLightboxForm.setWidth(Integer width) / getWidth() - sets/gets the width of the lightbox;
  • DHXExternalLightboxForm.setHeight(Integer height) / getHeight() - sets/gets the height of the lightbox;
  • DHXExternalLightboxForm.setClassName(String className) / getClassName() - sets/gets the name of a css class that will be applied to the lightbox;
  • DHXExternalLightboxForm.setView(String path) / getView - sets/gets the path to a view that you want to load.
