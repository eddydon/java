Timeline View Templates


Available templates:

  • DHXPlanner.templates.timeline_date(String format) - the date in the header of the view.
  • DHXPlanner.templates.event_class(String className) - the css style for the event container.
  • DHXPlanner.templates.timeline_scale_date(String format) - the items of the X-Axis.
  • DHXPlanner.templates.timeline_scale_label(String format) - the items of the Y-Axis.
  • DHXPlanner.templates.timeline_scalex_class(String className) - the css style for items of the X-Axis.
  • DHXPlanner.templates.timeline_scaley_class(String className) - the css style for items of the Y-Axis.
  • DHXPlanner.templates.timeline_second_scale_date(String format) - the items of the second X-Axis.
  • DHXPlanner.templates.timeline_second_scalex_class(String className) - the css style for the items of the second X-Axis.

Note, if the DHXPlanner.templates.setTimelineDate(String format) template isn't called, the date in the header will be set according to DHXPlanner.templates.setWeekDate(String format).


Cell mode personality


Available templates:

  • DHXPlanner.templates.timeline_cell_class(String className) - the css style for a cell.
  • DHXPlanner.templates.timeline_cell_value(String format) - the value presenting the number of scheduled events in a cell of the view.
  • DHXPlanner.Templates.timeline_tooltip(String text) - the tooltip over a cell containing some scheduled event(s).
