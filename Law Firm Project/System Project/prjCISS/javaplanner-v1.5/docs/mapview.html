Map view

 
By default, the left list of the view displays events beginning from the current date.

Initialization

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
DHXMapView view = new DHXMapView();
planner.views.add(view);

Main methods

  • DHXMapView.getViewType() - gets the type of the view.
  • DHXMapView.setELat(Double lat) / getELat() - sets/gets the latitude (in degrees) of the error position (the position that an event will take if it doesn't have associated coordinates).
  • DHXMapView.setELng(Double lng) / getELng() - sets/gets the longitude (in degrees) of the error position (the position that an event will take if it doesn't have associated coordinates).
  • DHXMapView.setErrorPosition(Double lat, Double lng) / getErrorPosition() - sets/gets the coordinates (in degrees) of the error position (the position that an event will take if it doesn't have associated coordinates).
  • DHXMapView.setFilter(DHXFilter filter) / getFilter() - defines filter for the view/gets the filter set for the view.
  • DHXMapView.setILat(Double lat) / getILat() - sets/gets the latitude (in degrees) of the initial map position.
  • DHXMapView.setILng(Double lng) / getILng() - sets/gets the longitude (in degrees) of the initial map position.
  • DHXMapView.setInitialPosition(Double lat, Double lng) / getInitialPosition() - sets/gets the coordinates (in degrees) of the initial map position.
  • DHXMapView.setLabel(String label) / getLabel() - sets/gets the text label.
  • DHXMapView.setMarkerGeoFail(String text) / getMarkerGeoFail() - sets/gets the text for the notice window that appears in case of unsuccessful geolocation response.
  • DHXMapView.setMarkerGeoSuccess(String text) / getMarkerGeoSuccess() - sets/gets the text for the notice window that appears in case of successful geolocation response.
  • DHXMapView.setName(String name) / getName() - sets/gets the name of the view.
  • DHXMapView.setTabClass(String className) / getTabClass() - specifies/gets the name of a css class that will be applied to the tab.
  • DHXMapView.setTabPosition(Integer offset) / getTabPosition() - sets/gets the right offset of the view tab in the tabbar. By default, tabs go right-to-left in the order of addition to the planner.
  • DHXMapView.setTabStyle(String style) / getTabStyle() - specifies/gets the style that will be applied to the tab.
  • DHXMapView.setTabWidth(Integer width) / getTabWidth() - specifies/gets the width of the tab.
  • DHXMapView.setStartDate(Date startDate) / getStartDate() - sets/gets the date that the view will display events beginning from. By default - the current date.
    • setStartDate(Calendar startDate)
    • setStartDate(Integer year, Integer month, Integer day)
  • DHXMapView.setEndDate(Date endDate) / getEndDate() - sets/gets the end date that the view will display events until.
    • setEndDate(Calendar endDate)
    • setEndDate(Integer year, Integer month, Integer day)

Details

Beware, to be correctly rendered on the map view data must contain a number of additional properties. The final list of mandatory properties (that data should have) looks as in:

  • start_date (Date or String) - the date when a task is scheduled to begin. The default format - ”%m/%d/%Y %H:%i”.
  • end_date (Date or String) - the date when a task is scheduled to be completed. The default format - ”%m/%d/%Y %H:%i”.
  • text (String) - the text of a task.
  • location - (String) the location of an event.
  • lat - (Double) the latitude of the event location.
  • lng - (Double) the longitude of the event location.
