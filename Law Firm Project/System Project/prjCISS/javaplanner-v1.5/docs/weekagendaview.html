WeekAgenda view

Initialization

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
DHXWeekAgendaView view = new DHXWeekAgendaView());//initializes the view
view.setLabel("Week");// the name of the tab
planner.views.add(view);//adds the view to the planner

Main methods

  • DHXWeekAgendaView.getViewType() - gets the type of the view. By default - 'week'.
  • DHXWeekAgendaView.setFilter(DHXFilter filter) / getFilter() - defines filter for the view/gets the filter set for the view.
  • DHXWeekAgendaView.setLabel(String label) / getLabel() - sets/gets the text label. The default value - 'Agenda'.
  • DHXWeekAgendaView.setName(String name) / getName() - sets/gets the name of the view.
  • DHXWeekAgendaView.setTabClass(String className) / getTabClass() - specifies/gets the name of a css class that will be applied to the tab.
  • DHXWeekAgendaView.setTabPosition(Integer offset) / getTabPosition() - sets/gets the right offset of the view tab in the tabbar. By default, tabs go right-to-left in the order of addition to the planner.
  • DHXWeekAgendaView.setTabStyle(String style) / getTabStyle() - specifies/gets the style that will be applied to the tab.
  • DHXWeekAgendaView.setTabWidth(Integer width) / getTabWidth() - specifies/gets the width of the tab.
