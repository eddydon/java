Views

The library provides 10 different views that allows you to present data in any possible way. A basic planner has 3 views: Day, Week and Month. Another views can be added (just like the default ones can be removed) as needed.

Default views

By default, the planner contains 3 tabs with the following views:

  • Day view
  • Month view
  • Week view

Additional views

Additionally, you can add:

  • Agenda view
  • Grid view
  • Map view
  • Timeline view
  • WeekAgenda view
  • Units view
  • Year view

Managing views

 
Watch out! Java Planner must have at least one view to run correctly.

Java Planner views are stored in a DHXPlanner.views collection.

Useful methods for working with views:

  • DHXPlanner.views.getViews() - returns a collection of planner views.
    ArrayList<DHXView> v = planner.views.getViews();
  • DHXPlanner.views.getView(Integer index) - returns a view with the specified index (zero-based numbering).
    DHXView firstView = planner.views.getView(0);
  • DHXPlanner.views.count()- returns the number of views in the collection.
    planner.views.count();
  • DHXPlanner.views.clear() - clears the DHXPlanner.views collection (removes all views from the planner).
    planner.views.clear();
  • DHXPlanner.views.add(DHXView view) - adds a new view to the planner collection.
    planner.views.add(new DHXAgendaView());
  • DHXPlanner. setInitialView(DHXView view) - makes the specified view selected initially (by default, it's the week view):
    planner.views.add(new DHXAgendaView());
planner.setInitialView("agenda");// "agenda" is the default name of the view
  • DHXPlanner.setInitialDate(Integer year, Integer month, Integer day) - sets the initial date.
    planner.setInitialDate(2009, 06, 01);
