DHXPlanner.templates

The library allows you to specify different templates. They can be used to change displaying dates and titles in the planner.

Common

To define the desired template, write it as it's stated in this documentation below, just changing DHXPlanner to the name of your DHXPlanner instance. 

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
planner.templates.setEventText("Subject: {text}");

As a result of this, a generated HTML page will contain the following code: 

planner.templates.event_text = function(start,end,ev){
   return 'Subject: ' + ev.text + '';
};

Referring to event properties

Note, within templates you can refer to the event properties as:

  • start_date - start;
  • end_date - end;
  • text - text.
DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
planner.templates.setEventBarText("{start:date(%d.%m.%Y)} - {end:date(%d.%m.%Y)}: {text}");

Setting the date format

You can use templates to set the desired format for the displayable dates. 

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
planner.templates.setDayScaleDate("{date:date(%d.%m.%Y)}");

The available format characters and combinations you can see here.

How is it used?

Generally, the format for a date is set as in: 

DHXPlanner.templates.set[templateName]("propertyName:date(someFormat)");

To know properties that a certain template disposes, check the input parameters of the related JavaScript function.

For example, you want to change the format of the date in the sub-header of the Day view:

Your actions:

  • First of all, you look up the desired template in the templates chapter.
    You go templates → Views-specific templates → Day view and find - DHXPlanner.templates.setDayScaleDate(String format).
  • Then, you follow the related link of the template and here it is, the default JavaScript definition:

    planner.templates.day_scale_date = function(date){
	return planner.date.date_to_str(planner.config.default_date);
};
  • You see that the function takes a parameter with the name date. So, you write:

    DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...       
planner.templates.setDayScaleDate("{date:date(%d.%m.%Y)}");
