Calendar Agenda View

Initialization

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
planner.views.add(new DHXAgendaView());//initializes the view
planner.setInitialView("agenda");//adds the view to the Java calendar control

Main methods

  • DHXAgendaView.getViewType() - gets the type of the calendar view.
  • DHXAgendaView.setLabel(String label) / getLabel() - sets/gets the text label.
  • DHXAgendaView.setName(String name) / getName() - sets/gets the name of the view.
  • DHXAgendaView.setTabClass(String className) / getTabClass() - specifies/gets the name of a css class that will be applied to the tab.
  • DHXAgendaView.setTabPosition(Integer offset) / getTabPosition() - sets/gets the right offset of the view tab in the tabbar. By default, tabs go right-to-left in the order of addition to the Java calendar.
  • DHXAgendaView.setTabStyle(String style) / getTabStyle() - specifies/gets the style that will be applied to the tab.
  • DHXAgendaView.setTabWidth(Integer width) / getTabWidth() - specifies/gets the width of the tab.
  • DHXAgendaView.setFilter(DHXFilter filter) / getFilter() - defines filter for the view/gets the filter set for the view.
  • DHXAgendaView.setStartDate(Date startDate) / getStartDate() - sets/gets the date that the view will display events beginning from. By default - the current date.
    • setStartDate(Calendar startDate)
    • setStartDate(Integer year, Integer month, Integer day)
  • DHXAgendaView.setEndDate(Date endDate) / getEndDate() - sets/gets the end date that the view will display events until.
    • setEndDate(Calendar endDate)
    • setEndDate(Integer year, Integer month, Integer day)
