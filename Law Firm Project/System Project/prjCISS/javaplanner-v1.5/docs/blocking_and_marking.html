Blocking, marking dates and pointer highlighting

Blocking and marking dates

Java Planner provides classes DHXBlockTime and DHXMarkTime that allow you to have some date range(s) blocked or/and marked.

Blocked/marked spans are contained in the DHXPlanner.timespans object. 

//blocks each Sunday
DHXBlockTime block = new DHXBlockTime();
block.setDay(DHXBlockTime.DHXDayOfWeek.SUNDAY);
planner.timespans.add(block);
 
//marks each SUNDAY with the 'blue_section' style
DHXMarkTime m1 = new DHXMarkTime();
m1.setDay(DHXMarkTime.DHXDayOfWeek.SUNDAY);
m1.setCssClass("blue_section");
planner.timespans.add(m1);

The duration of blocked/marked spans can be set in 2 ways (details of the methods used in the code snippets, see in the tables below):

 
setDays(), addZone() and setStartDate(), setEndDate() methods are used in pairs to set the blocking interval and can't be mixed and used in some other variation.
For example, you can't specify addZone(), setStartDate() or setDays(), setStartDate(), setEndDate() at the same time.

  1. With methods setStartDate(), setEndDate()

    //different days
DHXMarkTime m1 = new DHXMarkTime();
m1.setStartDate(2012,10,25);
m1.setEndDate(2012,10,27);
m1.setCssClass("blue_section");
planner.timespans.add(m1);
 
//or the same day from 8.00 to 20.00 
DHXMarkTime m1 = new DHXMarkTime();
m1.setStartDate(8, 0);
m1.setEndDate(20, 0);
m1.setCssClass("blue_section");
s.timespans.add(m1);
 
  2. With methods setDays(), addZone() 

    DHXMarkTime m2 = new DHXMarkTime();
m2.setDays(DHXBlockTime.DHXDayOfWeek.MONDAY, DHXBlockTime.DHXDayOfWeek.FRIDAY);
m2.addZone(new DHXZone(600, 900));// if zones aren't specified the entire day is blocked
m2.setCssClass("red_section");
planner.timespans.add(m2);

The remaining methods can be used in any variant.

Pointer highlighting

Pointer highlighting is an ability of the planner to highlight the area under the pointer.

The ability is provided by the DHXPlanner.highlight class which has the following methods:

  • setCreateOnClick(Boolean createOnClick) - if the method is called with the true value, on a single click the user can create events with the duration = highlighted area;
  • setCssClass(String className) - the name of a css class that will be applied to the highlighted area;
  • setEnabled(Boolean isEnabled) - checks whether the pointer highlighting is activated (if it is - returns true);
  • setFixedStep(Boolean enableMoving) - enables/disables moving the highlighted area by step units (i.e. you won't be allowed to move along the full time scale);
  • setHTML(String text) - the HTML content that will be added to the highlighted area;
  • setStep(Integer step) - the duration of the highlighted area in minutes.

To enable the pointer highlighting, call method DHXPlanner.highlight.enable(String className, Integer step): 

planner.highlight.enable("highlight_area", 30);

The DHXPlanner.highlight.enable() method can be called in 2 ways:

  1. DHXPlanner.highlight.enable(String className);
  2. DHXPlanner.highlight.enable(String className, Integer step);
  • css - the name of a css class that will be applied to the highlighted area.
    Note, to set colors or borders for the area you must use the :hover pseudo-class.

    <style>
        .highlight_area {
            opacity: 0.3;
            z-index:0;
            filter:alpha(opacity=30);
        } 
        .highlight_area:hover {
            background-color: #99F799;
        }
</style>
  • step - the duration of the highlighted area in minutes. The default value - 60.
