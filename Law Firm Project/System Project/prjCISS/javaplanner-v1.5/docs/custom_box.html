Custom display for events

If the standard look of the events boxes don't fit you, you can provide your custom display.

Generally, a common technique for customizing events boxes consists in the following:

  1. Create a DHXEventBox object.
    The DHXEventBox class is a general class for custom event boxes.

  2. Define a template for the new box.
    The template is specified through the DHXEventBox.setTemplate(String template) method

  3. Define and set a css class for the new box.
    The name of a css class is set through the DHXEventBox.setCssClass(String className) method

  4. Set the new box for events.
    A new box for events is specified through the DHXPlanner.templates.setEventBox(DHXEventBox ev) method. If you don't call the method, events will be rendered in the standard(default) boxes.


The DHXEventBox class provides 4 important methods:

  1. DHXEventBox.setCanDrag(Boolean canDrag) - specifies whether the user can drag the event box;
  2. DHXEventBox.setCanResize(Boolean canResize) - specifies whether the user can change the duration of the event by dragging it onto the planner;
  3. DHXEventBox.setCssClass(String className) - sets the name of a css class that will be applied to the event box;
  4. DHXEventBox.setTemplate(String template) - sets the template for the event box. The template is defined in a usual way (read more on templates here). The template function takes 3 parameters: ev, start, end (details).

If you don't want to use the default 'drag', 'resize' controls you should call the DHXEventBox.setCanDrag(Boolean canDrag) and DHXEventBox.setCanDrag(Boolean canDrag) methods with the false value
and define your custom controls in the template function (DHXEventBox.setTemplate(String template)).
In this case, remember that some styles have a special purpose and must go first:

  • .dhx_event_move - an element with this style can be dragged (usually, it's the event header);
  • .dhx_event_resize - dragging an element with this style will change the event duration.

Below is an example of some custom event box: 

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
DHXEventBox ev = new DHXEventBox();
ev.setTemplate("<div class='custom_event' style='background-color:#1796B0; opacity:0.7; padding:10px;'>"+
          "<span class='event_date'>{start_date:date(%H:%i)}</span> - "+
          "<span class='event_date'>{end_date:date(%H:%i)}</span><br>"+
          "<span><b>{text}</b></span><br>"+
          "</div>"+
          "</div>");
ev.setCssClass("custom_event");
planner.templates.setEventBox(ev);
