Defaults views

By default, the planner has 3 views:

  • Day view
  • Week view
  • Month view

Initialization

You needn't to write any code to initialize the stated views.

To refer to one of the views, you should use DHXView firstView = planner.views.getView(i), where

  • i=0 ⇒ month view
  • i=1 ⇒ week view
  • i=2 ⇒ day view
DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
DHXView monthView = planner.views.getView(0);
monthView.setLabel("Month calendar"); // sets the label for the tab of the month view

Main methods

  • DHXView.getViewType() - gets the type of the view.
  • DHXView.setFilter(DHXFilter filter) / getFilter() - defines filter for the view/gets the filter set for the view.
  • DHXView.setLabel(String label) / getLabel() - sets/gets the text label.
  • DHXView.setName(String name) / getName() - sets/gets the name of the view.
  • DHXView.setTabClass(String className) / getTabClass() - specifies/gets the name of a css class that will be applied to the tab.
  • DHXView.setTabPosition(Integer offset) / getTabPosition() - sets/gets the right offset of the view tab in the tabbar. By default, tabs go right-to-left in the order of addition to the planner.
  • DHXView.setTabStyle(String style) / getTabStyle() - specifies/gets the style that will be applied to the tab.
  • DHXView.setTabWidth(Integer width) / getTabWidth() - specifies/gets the width of the tab.

Limiting the number of events per cell in the Month view

By default, the calendar increases the height of a cell to show all the assigned events in it. But in case of a big number of events, cells can be very long and look overcrowded with text.
Starting from version 1.5, you have a possibility to control the number of events displayed in a cell, i.e. the height of cells.

To set the maximum number of events per cell, use the DHXPlanner.config.setMaxMonthEvents method: 

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
 
planner.config.setMaxMonthEvents(3);

If the number of assigned events is more than the value you've set, the calendar will display the 'View more' link. The link will follow the user to the Day view where he or she can see a full list of assigned events.

To customize the 'View more' link, you can use the DHXPlanner.templates.setMonthEventsLink template: 

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
planner.templates.setMonthEventsLink("<a>View more({count} events)</a>");
