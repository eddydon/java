Compatibility

Browsers

  • Internet Explorer 7+ (standard mode only);
  • Firefox 3+;
  • Safari 3+;
  • Opera 9+;
  • Google Chrome.

Interferences (CSS)

Sometimes Java Planner can interfere with your web-site code and cause clashes in style sheets (CSS).

As far as Java Planner has an open source client-side JavaScript API, these interferences can be quickly fixed from the code. If you have faced this problem, check out our Forum.
