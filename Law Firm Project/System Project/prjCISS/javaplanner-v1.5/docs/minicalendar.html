Date Picker (mini calendar)

MiniCalendar provides ability to render a small month view in any html container on a page.

MiniCalendar can be put into 3 places (each place 'requires' its specific code):

  • in the planner header

    planner.calendars.attachMiniCalendar();

  • in the lightbox

    planner.lightbox.add(new DHXLightboxMiniCalendar("cal", "Time period"));

  • in any html container outside the planner

    DHXMiniCalendar cal = new DHXMiniCalendar("container_id");
cal.setNavigation(true);
planner.calendars.add(cal);

Initialization

Mini calendar in the header

To add miniCalendar to the header, call method DHXPlanner.calendars.attachMiniCalendar(): 

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
planner.calendars.attachMiniCalendar();

The method returns a DHXMiniCalendar object that can be used for further calendar configuration. 

DHXMiniCalendar cal = planner.Calendars.AttachMiniCalendar();
cal.setNavigation(true);

Mini calendar in the lightbox

If you want to use the mini calendar in the details form for start and end dates selection, just add the appropriate control - a DHXLightboxMiniCalendar object: 

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
planner.lightbox.add(new DHXLightboxMiniCalendar("name", "label"));

Mini calendar outside the planner

To append the mini calendar to an html container outside the planner, you should use the following code: 

DHXPlanner planner = new DHXPlanner("./static/", DHXSkin.TERRACE);
...
DHXMiniCalendar cal = new DHXMiniCalendar("container_id");// initialization
cal.setNavigation(true);// configuration
planner.calendars.add(cal);// adding to the page

Note, the container for the mini calendar can be specified in 2 ways:

  1. Directly in the constructor (as shown in the example above);
  2. After initialization (through the DHXMiniCalendar.setContainer(String container) method):

    DHXMiniCalendar cal = new DHXMiniCalendar();
cal.setContainer("container_id");
planner.calendars.add(cal);

Methods

  • DHXMiniCalendar.setContainer(String containerId) - the id of a DOM element that the mini calendar will be placed to.
  • DHXMiniCalendar.setDate(Calendar initialDate) - the initially selected date.
  • DHXMiniCalendar.isAttachedToHeader(Boolean isAttached) - checks whether the mini calendar is attached to the planner or another DOM element.
  • DHXMiniCalendar.setNavigation(Boolean isScrolled) - allows mouse scrolling.
  • DHXMiniCalendar.setPosition(String className) - the css class name of the element that the mini calendar will be attached to. The method is used to set the position of the calendar in the planner. Specify the desired html element through its css class and anytime you click on this element, the mini calendar will appear.
