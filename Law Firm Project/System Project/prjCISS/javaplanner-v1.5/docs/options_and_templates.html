Configuration classes

Java Planner is a flexible library and gives developers the possibility to customize most of the calendar's appearance. One of the means that provides such flexibility is a set of 3 configuration classes:

  • DHXPlanner.config- contains configuration methods for dates, scale, controls etc.
  • DHXPlanner.templates - contains template functions for dates and titles used in the planner, tooltips and styling.
  • DHXPlanner.xy - provides methods for sizing planner's elements.
